# FSharp.Data Template

## Restore after clone
```bash
dotnet tool restore
dotnet paket restore
```

## Run
```bash
dotnet fsi --exec src/Main.fsx
```

## Init Commands Memo

```bash
$ dotnet new tool-manifest
$ dotnet tool install paket
$ dotnet paket init
$ # edit paket.dependencies
$ dotnet paket restore
```
